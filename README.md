# AI & AR Lab

Первая рабочая версия образовательной PWA-платформы по авторской программе учителя информатики Анны Валерьевны Колтуновой **«Применение искусственного интеллекта и дополненной реальности в проектной деятельности»**.

## Что реализовано

- Современная адаптивная главная страница с описанием программы, шестью модулями, списком из 34 уроков, результатами обучения, блоком автора и прогрессом.
- Полноценный урок №1: теория, интерактивная схема, временная линия, сферы применения, практическое задание, мини-игра, тест из 8 вопросов, рефлексия и завершение урока.
- Интерактивная презентация из 10 слайдов с клавиатурной навигацией, свайпами, счётчиком и полноэкранным режимом.
- Автоматическое сохранение практики, рефлексии, лучшего результата теста и завершения урока в `localStorage`.
- Цифровой значок и поздравительная анимация после завершения урока.
- Базовая PWA: манифест, иконка и service worker с офлайн-кешем оболочки приложения.
- Для всех 34 уроков созданы отдельные страницы с последовательной навигацией и комплектом материалов: КСП, презентация, рабочий лист, видео, практика и рефлексия. Пустые уроки отмечены статусом «Скоро».

## Локальный запуск

Сборка и установка зависимостей не нужны. Для корректной работы service worker откройте проект через локальный HTTP-сервер:

```bash
python3 -m http.server 8080
```

Затем перейдите на [http://localhost:8080](http://localhost:8080).

> Не рекомендуется открывать файлы через `file://`: браузеры не регистрируют service worker для такого протокола.

## Публикация на Netlify

1. Подключите репозиторий в Netlify.
2. Оставьте поле **Build command** пустым.
3. Укажите **Publish directory**: `.`.
4. Запустите публикацию.

Проект полностью статический и не требует переменных окружения или внешних API.

## Управление презентацией

- `←` и `→` — предыдущий и следующий слайд;
- `Пробел` — следующий слайд;
- кнопка `⛶` — полноэкранный режим;
- на сенсорном экране доступны свайпы влево и вправо.

## Хранение данных

Данные сохраняются только на устройстве пользователя в ключе `aiArCourseProgress` в `localStorage`. Авторизация и передача данных на сервер не используются.
